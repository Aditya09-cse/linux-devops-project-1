# Linux DevOps Practice Project

This repository contains hands-on Linux and DevOps practice performed on an Ubuntu server.

## Project Overview
In this project, I worked on a simulated production environment and practiced:

- User and group management
- File and directory permissions
- Log monitoring and cleanup
- Process management
- Backup and restore
- Automation using shell scripting
- Cron jobs
- System health checks and troubleshooting

## Automation
- Created a shell script to archive log files
- Automated log cleanup using cron
- Implemented proper logging for cron jobs

## Tools & Technologies
- Linux (Ubuntu)
- Bash scripting
- Cron
- Git & GitHub
- AWS EC2

## Key Learning
This project helped me understand real-world Linux server administration and DevOps fundamentals.
